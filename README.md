# Foundation-Project: Online job portal for student
-Before using any feature for a specific actor, you must create an account and log in with an account that has that role; only then can you use that actor’s functions.
-If you want to create an employer account, go to the admin dashboard to create it.
-To create a post, you must first fill in the required information in the Edit Profile section (including the company avatar for the posted job).

*Student function:
-Sign up
-Log in
-Search for jobs
-Create a CV
-Download the CV
-Apply for a job
-View basic information about the hiring company
-Edit profile
-Receive notification

*Employer funtion:
-Log in
-Post a job
-Manage job posts
-Manage candidates (manage applicants’ CVs)
-Edit profile
-Receive notifications
